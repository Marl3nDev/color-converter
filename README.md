# Конвертер цветов

Простое React-приложение для конвертации цветов из HEX в RGB.

## Функции

- Ввод кода цвета в формате HEX (например, #9921ff)
- Отображение эквивалента в RGB
- Изменение цвета фона страницы
- Показ ошибки для недействительных кодов HEX

## Демо
Посмотреть демо: [GitHub Pages](https://marl3ndev.github.io/color-converter/)

## Начало работы

1. Установите зависимости: `npm install`
2. Запустите приложение: `npm start`
3. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## Сборка

Запустите `npm run build` для сборки приложения в продакшн.
